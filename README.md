# UI预览工具

一个简单的UI素材预览工具，用于查看和测试UI组件。

## 项目结构

```
├── index.html        # 主预览页面
├── simple.html       # 简化版预览
├── test.html         # 测试页面
├── reference.png     # 参考图片
├── assets/           # UI素材资源
│   ├── background/   # 背景图片
│   ├── desktop/      # 桌面素材
│   ├── statusbar/    # 状态栏素材
│   └── statusbar_shadow/ # 状态栏阴影素材
└── README.md         # 项目说明
```

## 功能特性

- 多页面预览：提供多种UI布局预览
- 资源管理：集中管理UI素材资源
- 快速测试：便于UI组件的快速验证

## 使用方法

1. 打开 `index.html` 查看主预览页面
2. 打开 `simple.html` 查看简化版预览
3. 打开 `test.html` 进行功能测试

## 开发说明

这是一个纯前端项目，不需要构建工具或依赖安装。

## 许可证

本项目仅供学习和参考使用。
